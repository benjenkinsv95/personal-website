The Boston Scavenger provides Boston locals with up-to-date information about excellent, food trucks nearby. It was a group project for my Agile Software Development class that we completed in 3 sprints.

The sprints closely followed The Scrum Guide, were managed on Trello, and documented on the project's GitHub.

To enable quick feedback on UI changes, we set up a CI/CD pipeline. The CI/CD pipeline allows the most recent successful build to be available online at any time.
